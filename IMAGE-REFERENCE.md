## HOW TO ADD AN IMAGE

1. Export your screenshot from Figma (or take a screen capture)
2. Rename it to match the filename in this table exactly
3. Drop it into this folder (assets/images/)
4. Upload to GitHub — the image will appear on the site automatically

## TIPS

- PNG is best for screenshots and UI designs
- JPG is fine for photos (e.g. user testing session photos)
- GIF works for animated demos — or use a static PNG if you don't have the GIF
- Recommended width: 1400–2000px for full-width images, 1000px for medium
- If an image doesn't show, double-check the filename matches exactly (case sensitive)
